## 🚀 使用方法

### 直接打开
双击 `index.html` 文件，或在浏览器中打开：
```
file:///c:/Users/Administrator/copy/音乐软件2/web/index.html
```

### 使用本地服务器（推荐）
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server

# 然后访问: http://localhost:8000
```
