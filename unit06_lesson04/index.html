深入学习 JavaScript 核心知识与 DOM 操作

📖 一、JavaScript 基础语法

💡 核心概念：JavaScript 是网页的"行为层"，负责实现动态交互效果。它与 HTML（结构）、CSS（样式）共同构成现代网页的三大支柱。

（一）变量声明

变量就是存储信息的地方，类似于一个盒子，里面可以放入各种信息（如文字、数字、对与错）。我们使用 let 或 const 来定义一个变量：

  • let 就像一个可重复使用的储物柜，里面的东西可以换，用来声明可修改的变量
  • const 就像一个保险箱，一旦锁好里面的东西就不会变，用来声明不可改变的变量

使用 let 和 const 声明变量

关键字 代码示例 说明
let let name = "zhugezhangsan" ;
let age = 18 ; 		声明可变变量
可以存储字符串、数字等类型
变量的值可以后续修改
const const PI = 3.14159 ;
const siteName = "我的网站" ; 		声明常量
一旦赋值后不可修改
适用于固定不变的值

（二）数据类型

Number

42, 3.14

String

"Hello", '世界'

Boolean

true, false

Null / Undefined

null, undefined

数据类型示例

变量名 赋值 数据类型 说明
isStudent let isStudent = true ; Boolean 布尔值：表示是学生
score let score = 98 ; Number 数字类型
nickname let nickname = "Sunny" ; String 字符串类型

（三）基本运算符

我们可以让 JavaScript 做加法、减法、乘法和除法，或者比较两个数是否相等、大小等，就像一个计算器，也像一个判断器，可以做数学计算，也能做逻辑判断。

运算符示例

运算类型 代码示例 说明
加法 let sum = 5 + 3 ; 加法，结果是 8
减法 let diff = 10 - 4 ; 减法，结果是 6
相等判断 let isEqual = ( 5 == 5 ); 比较是否相等，结果是 true
大小比较 let isGreater = ( 8 > 10 ); 比较大小，结果是 false

（四）函数定义

函数是一段可以重复使用的代码，就像一个"工具"或"按钮"，每次调用它，它就会自动执行里面的任务。可以借助"自动贩卖机"来理解：你输入名字（投币），它输出一段话（饮料），我们可以通过它来做一些不需要变化的固定事情。

函数是可重复使用的代码块

步骤 代码 说明
1. 函数定义 function greet (name) {
     return "Hello, " + name;
} 		定义一个名为 greet 的函数
接收一个参数 name
返回拼接后的问候语
2. 调用函数 console. log ( greet ( "Alice" )); 调用函数并传入参数 "Alice"
输出结果：
Hello, Alice

🔧 二、JavaScript 与网页：DOM 操作

💡 什么是 DOM？

  • DOM（Document Object Model，文档对象模型）像是网页的"骨架"，它把网页上的每一部分都当成一个可以"触摸、改动"的对象
  • 网页上的文字、图片、按钮、链接都可以被看作是一个"积木块"，而 JavaScript 就是用来操作这些"积木"的工具
  • DOM 是网页的"内部骨架"，JavaScript 就像一只"手"，可以去拿掉、替换、增加这些骨架上的各个骨头

📊 DOM 树结构示意图

（一）获取网页上的内容

要操作网页上的某个部分，第一步是找到它。我们可以使用浏览器提供的"查找命令"来获取内容，document 就像是整个网页的"地图"，这些方法就像是在说："帮我找到那个地方的那个内容。"

方法 作用 示例
getElementById() 根据元素的 id 名称 找到网页上的内容 document.getElementById("header")
querySelector() 根据CSS的写法找到网页上的内容 document.querySelector(".title")

选择元素示例代码

变量名 代码 说明
header let header = document. getElementById ( "header" ); 找到 id 是 header 的部分
title let title = document. querySelector ( ".title" ); 找到 class 是 title 的标题

（二）修改元素内容

一旦找到网页的内容后，我们可以对它进行修改，比如——改变文字、改变颜色、甚至让它"动起来"。

操作 属性 示例 效果
改文字 innerText title.innerText = "欢迎" 修改标题文字内容
改样式 style header.style.color = "red" 修改背景颜色
改图片 src img.src = "new.jpg" 替换图片内容

修改元素示例代码

操作类型 代码 说明
修改文本内容 element. innerText = "新内容" ; 将元素的文字修改为"新内容"
修改文字颜色 element.style. color = "red" ; 将元素的文字颜色改为红色
修改字体大小 element.style. fontSize = "20px" ; 将字体大小设置为 20 像素
修改图片 image. src = "new-image.jpg" ; 替换图片为新的图片文件

⚙️ 三、DOM 操作：修改网页内容

（一）添加元素

想象网页就像一个房间，你可以"放进"新的家具。JavaScript 里，我们可以创建新的网页内容（比如文字、图片、按钮等），再把它放进页面。

语法流程：创建新元素 → 写入内容 → 放入网页

动态创建新元素并添加到页面

步骤 代码 说明
1. 创建元素 let newDiv = document. createElement ( "div" ); 创建一个新的 <div> 区块
2. 写入内容 newDiv. innerText = "这是一个新添加的内容" ; 向新元素写入文字内容
3. 添加到页面 document.body. appendChild (newDiv); 把这个区块放到网页的最后

效果：执行后，网页底部就会多出一句"这是一个新添加的内容"

（二）删除元素

如果网页中我们不再需要某个内容，我们可以把它"拿掉"，这就像把桌上的便签纸撕掉一样简单。

从页面中移除不需要的元素

步骤 代码 说明
1. 找到元素 let oldDiv = document. getElementById ( "oldDiv" ); 找到要删除的元素
2. 删除元素 oldDiv. remove (); 从页面移除

效果：网页上名字是 oldDiv 的部分会被移除

（三）简单的交互示例：点击按钮改变背景颜色

在网页中，点击按钮改变样式是最常见的交互之一，比如：点击"切换主题"改变网页背景色、点击"高亮"让重要内容突出显示、点击"夜间模式"让页面变暗。下面我们通过一个简单的例子来学习如何修改元素的背景颜色和文字内容。

示例：点击按钮，色块在青色和紫色之间切换

部分 代码 说明
HTML 色块 <div id =" colorBox " style ="..." >
  这是一个色块
</div> 		展示区域：一个带背景色的色块
HTML 按钮 <button id =" changeBtn " > 改变颜色 </button> 切换按钮
JS 获取元素 let button = document.getElementById("changeBtn");
let box = document.getElementById("colorBox"); 		找到按钮和色块元素
JS 切换逻辑 button.addEventListener("click", function() {
  if (条件为青色) {
    box.style.backgroundColor = "#a855f7";
  } else {
    box.style.backgroundColor = "#06b6d4";
  }
}); 		监听点击事件，在青色和紫色之间切换

效果说明：点击按钮，色块会在青色（#06b6d4）和紫色（#a855f7）之间来回切换，同时文字内容也会改变

💡 知识点：这个例子同时展示了修改样式（style.backgroundColor）和文字（innerText）两个重要的 DOM 操作

🎯 四、事件驱动：响应用户行为

💡 什么是"事件"？

"事件"其实就是用户在网页上做的动作，比如——点击一个按钮、把鼠标移上去、在输入框里打字、滚动页面。

当这些动作发生时，JavaScript 就能"听到"，并根据你的指令"做出反应"。

举例说明：网页就像一位"服务员"，你的一举一动（点击、输入、滑动）都是"事件"，而 JavaScript 就是告诉这位服务生："当客人做了某件事，要怎么回应他"。

（一）常见事件类型

click

鼠标点击

mouseover

鼠标悬停

keydown

按下键盘

scroll

页面滚动

（二）绑定事件监听器

🔄 JavaScript 交互流程示意图

我们用 addEventListener() 这个函数来告诉网页："当某个事件发生时，就要执行另外一件事"。

形象理解：在酒店门口安排一个礼仪人员（监听器），当"有人走过来"时，礼仪人员就立刻执行"开门"的动作（函数）。

语法格式：

元素.addEventListener("事件类型", 要执行的操作);

示例：点击按钮后弹出提示

步骤 代码 说明
1. HTML 按钮 <button id =" helloBtn " > 点我一下 </button> 创建按钮
2. 获取按钮 let button = document. getElementById ( "helloBtn" ); 找到按钮元素
3. 监听事件 button. addEventListener ( "click" , function () {
    alert ( "你好！按钮被点击了～" );
}); 		监听点击事件，弹出提示

效果：点击按钮后，网页弹出一个小提示框，显示"你好！按钮被点击了～"
