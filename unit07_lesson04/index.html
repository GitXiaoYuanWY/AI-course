📊 一、JSON数据格式回顾

（一）JSON的作用

  • JSON是一种通用的数据格式，几乎所有API、HTTP请求、AI模型都使用它来传递信息。
  • 在N8N中，所有节点之间传递的数据都是JSON结构。每个节点输出的数据，都会以"键值对"的形式流向下一个节点。

（二）JSON的语法

1

基本结构

  • JSON的数据都包裹在一对大括号 {} 中，内部由一组 "键（key）: 值（value）" 组成。
  • 多个键值之间用逗号分隔。

案例：

{"name":"张三","age":"24"}
  • "name" 是键（key）
  • "张三" 是值（value）
  • 多个键值之间用逗号分隔
2

可以嵌入JSON

JSON可以嵌套另一个JSON

{
  "student": {
    "name": "李四",
    "age": 21,
    "major": {
      "name": "计算机",
      "direction": "AI"
    }
  }
}

这说明每个"键"下面可以继续展开一层"结构化数据"。在API返回的数据中，嵌套结构非常常见。

JSON嵌套结构可视化

3

如何使用上一个节点输入的JSON数据？

  • {{$json.name}} 会返回 "张三"
  • {{$json.student.name}} 会返回 "李四"
  • {{$json.student.major.name}} 会返回 "计算机"

备注：只需要理解原理即可，不需要自己动手写

（三）List（列表）

List的含义

List是由一组数据组成的有序集合，用方括号 [] 表示。

List的语法规范

  • 用 [] 来表示
  • 案例：["苹果", "香蕉", "橙子"]

JSON中的key（键）的value（值）可以是列表list

案例表示：

{
  "students": ["张三", "李四", "王五"]
}

list列表中的对象也可以是JSON

案例表示：

{
  "students": [
    {"name": "张三", "score": 90},
    {"name": "李四", "score": 85}
  ]
}

这里的students的值就是一个列表，也就是list。

在N8N中的术语：

  • 一整个list（item）可以看作一个表格
  • key（键）就是表格中的每一列
  • value（值）代表表格中每一行
  • items就是表格中的每一条记录

（四）N8N中节点读取数据的方式

1

Fixed（固定值）

  • 手动输入具体的内容，相当于JSON格式中的"value（值）"，也就是固定死了你要输入的数据
  • 例如：姓名 = "张三"
2

Expression（表达式）

  • 通过 {{ }} 引用上一个节点输出的数据，相当于JSON格式中的"key（键）"
  • 例如：姓名 = {{$json.name}}
  • Expression让工作流"动起来"，实现动态数据传递。

🔧 二、数据处理节点

N8N提供了大量专门处理数据的"进阶节点"，
它们能帮你实现筛选、聚合、合并、判断、转换等功能。

数据处理节点流程示意图

（一）常规节点

Edit Fields (Set)

功能：添加、修改或删除字段。

（二）增加/移除项目节点

Filter（过滤）

功能：根据某一个条件，筛选出符合条件的数据

举例：只保留成绩在80分以上的学生

Limit（限制）

功能：限制输出的数量。

举例：只输出前10条数据，或者是只输出后十条

Split Out（拆分）

功能：将一个item拆分成多个items

（三）合并项目节点

Aggregate（聚合，反向的split out）

功能：可以将多个数据组合成一个，也就是将多个items组合成一个item

Merge（合并）

功能：将两组数据合并在一起。

方法
Append（叠加）

把两张表不做任何修改的拼接在一起；把所有的items对应叠加

Combine（匹配）

按照同一个"键":"值"对齐拼接

SQL语言（偏向客制化）

用查询语句客制化合并逻辑

Choose Branch（选择输出）

选择用哪一个键来输出

对比Aggregate与Merge节点的区别

  • Aggregate：将同一数据流中的多个items合并为一个item
  • Merge：将两个不同数据流的数据进行合并

（四）判断项目节点

IF

功能：是最常用的逻辑判断节点，当数据符合某个条件时，走"A路径"；否则，走"B路径"。

举例：淘宝购物：如果付款成功 → 发货； 如果付款失败 → 提醒重新支付。

（五）code节点

Code

功能：Code节点是"自由区"，可编写JavaScript脚本处理复杂逻辑。

它能实现：批量计算、字符串拼接、数据格式转换等高级功能。

案例应用
  • 使用code（java）节点将字符串转换成数组
  • 使用code（java）节点进行数据的排列
  • 使用code（java）节点对格式进行转换

（六）格式转变节点

节点 功能 实际用途
Compression 压缩/解压文件 报告归档、日志打包
Convert to File 生成可下载文件 AI生成文案导出txt
Crypto 加密/解密数据 保护敏感内容
Edit Image 编辑图片 批量缩放、裁剪封面图
Extract from File 提取文件内容 从Excel中读取表格
HTML 转换为网页格式 自动生成邮件正文
Markdown 转换为笔记格式 输出到Notion或知识库
XML 与旧系统数据互转 兼容传统API系统
