01

Trae平台介绍

Trae平台概述

1. Trae的定义与定位

Trae是由字节跳动推出的一款AI原生集成开发环境（AI IDE），专为AI辅助编程与开发而设计。与传统IDE（如Visual Studio Code或WebStorm）不同，Trae在系统架构中嵌入了大语言模型，让使用者可以不敲代码，只输出自己的想法就能进行开发。

这种设计使开发者能够通过文字描述、语义指令或图片输入的方式直接生成网页代码，从而显著降低网页开发的技术门槛。

核心特性说明

（1）AI原生编程能力

Trae支持用户通过自然语言描述需求，由AI生成符合语义的前端代码。例如，输入"生成一个包含导航栏和图片展示区的旅游网站主页"，系统即可自动创建HTML结构、CSS样式及相应的脚本逻辑。

（2）多模态交互设计

除文字指令外，Trae还支持通过上传设计草图、界面截图等图像输入，AI会自动识别界面布局并生成对应前端结构。

（3）深度集成多模型体系

Trae内置多种顶级大模型，允许用户根据任务需求选择不同模型执行。例如，Claude 3.5适用于语义分析与逻辑生成，GPT-4o适用于视觉输入与代码结构生成。

（4）Builder自动化项目构建

Builder模式能自动识别项目结构，快速生成文件组织、依赖配置与构建脚本。该特性适合教学中快速生成标准化项目模板，提高学生练习效率。

（5）完全免费与云端同步

Trae作为字节跳动官方推出的AI IDE，提供永久免费使用，并支持云端项目同步与共享，便于多设备协作与教学示范。

Trae的应用优势

（1）低门槛学习环境

无需复杂配置即可开始网页开发，新手也能快速上手，让学习者将精力集中在网页设计本身而非环境搭建。

（2）实时效果预览

每次编辑后可立即查看网页呈现效果，所见即所得的开发体验让调试和优化变得直观高效。

（3）智能提示与纠错

AI自动检测语法与逻辑错误，并提供修改建议，有效减少代码错误，加速问题定位与解决。

（4）教学友好性高

集教学、演示、实验于一体的集成环境，适合课堂教学与自主学习，提供完整的学习支持体系。

下载与安装流程

1. 访问官方网站

可通过浏览器访问Trae官方网站（trae.cn），根据操作系统（Windows、macOS或Linux）选择相应版本下载。

下载".dmg"文件后双击打开，将Trae图标拖入"应用程序"文件夹即可完成安装。首次运行可能需在系统隐私设置中允许应用打开。

熟悉Trae界面

主界面布局与功能区划分

Trae的主界面采用模块化布局，在Trae文件夹内新建一个文件夹，在Trae内打开新建的文件夹，即可看见常用的五大功能区：

五大功能区

  1. 顶部菜单栏：包含项目、编辑、视图、工具与帮助等功能
  2. 侧边栏：用于项目文件管理、版本控制与搜索文件等功能
  3. 主编辑区：核心代码编辑窗口，支持多标签页
  4. AI面板：位于右侧，用户可以使用自然语言和AI进行交互
  5. 底部控制台：包含终端输出、错误信息与运行日志

核心功能区详细说明

1. AI面板功能

AI面板是Trae的核心创新区域。用户可以在右侧和AI的对话框内输入自然语言指令，让AI生成网页结构、优化样式或解释代码逻辑。AI面板可执行以下任务：

  • 生成HTML、CSS、JavaScript代码段
  • 对选中代码提供解释与改进建议
  • 调试时根据错误信息自动推荐修复方案
  • 支持语音或图片输入

实际演示

例如输入"创建一个响应式导航栏"，AI会在几秒钟内返回完整的HTML和CSS片段。

2. 编辑区功能

编辑区支持语法高亮、智能提示与实时预览，兼容多种语言（HTML、CSS、JS、Python等）。其最大特点在于双窗口预览机制：一侧为代码编辑，一侧实时显示网页渲染结果。

网页基础概念

网页的组成与工作原理

1. 网页的定义

网页是互联网信息传播的最常见的方式，是由HTML、CSS和JavaScript等技术组成的文档。

  • HTML负责网页的内容与结构
  • CSS负责控制网页的样式与布局，类似于手机的"个性主题"
  • JavaScript负责实现网页的交互与动态逻辑，比如点击按钮弹出提示框

2. 浏览器渲染网页内容的过程

浏览器在加载网页时，会执行以下步骤：

  1. 解析HTML文件，构建DOM（文档对象模型）
  2. 解析CSS文件，生成样式规则树
  3. 合并DOM与样式树，形成渲染树
  4. 执行JavaScript以更新文档内容或样式
  5. 将最终布局绘制到屏幕上

3. HTML基本结构

以下是一个典型网页的基础框架：

<!DOCTYPE html>
<html>
<head>
   <title>我的第一个网页</title>
</head>
<body>
   <h1>欢迎来到我的网页</h1>
   <p>这里展示网页的基础结构。</p>
</body>
</html>

其中：

  • <head> 部分定义网页信息（如标题、编码、样式引用）
  • <body> 部分展示页面内容（文字、图片、链接等）
05

用AI创建网页

项目创建：与AI对话生成网页

在Trae中，无需手动新建文件或设置模板。只需在对话框中向系统输入指令，例如：

"帮我创建一个个人简介网页，包含姓名、专业、兴趣介绍。"

AI即可自动生成完整项目结构，包括：

  • index.html：主页面结构
  • style.css：样式文件
  • script.js：交互逻辑文件

系统会在几秒内构建好网页并展示预览弹窗，点击预览即可查看AI生成好的个人简介网页。

理解网页结构：AI生成的HTML示例

虽然项目由AI自动生成，但理解网页结构依然重要。理解这些标签的作用，有助于在后续课程中修改和扩展AI生成的网页。

实践项目：创建并优化个人简介网页

项目目标

创建一个展示个人基本信息的网页，并通过AI优化功能慢慢提升页面质量。

实现过程

  1. 在Trae对话框中输入指令："创建一个简洁的个人简介网页，主题为蓝白色。"
  2. 等待AI生成网页并预览
  3. 选中部分代码，使用"让AI解释"功能理解其作用
  4. 故意制造一个错误（如删除一个结束标签），再让AI帮忙修复
  5. 修改提示词，如"添加一张头像图片"或"调整排版为两列布局"，观察生成效果

思考与拓展

  • 不同提示词对生成结果有何影响？
  • 你能让AI帮你添加新模块或换一种视觉风格吗？
06

AI辅助功能

AI辅助功能：理解与改进网页

Trae的AI不仅能生成网页，还能对内容进行解释、优化和修复，帮助学习者更快掌握网页知识。

1. 界面与代码解释

选中任意需要AI解释的内容，选择"添加到对话"（快捷键Ctrl+U）即可在右边和AI的对话框中看见所选取的内容，再输入如"解释这段代码"等语句给AI，AI能直接返回详细说明。

这种功能适合初学阶段理解代码含义以及网页的布局逻辑。

2. AI修复功能

当网页显示异常或控制台报错时，AI会自动根据错误提示输出修复建议，并且会自主完成修复操作，脱离手动敲代码的阶段。

AI可以快速检测错误并说明原因，显著减少排错时间，提高学习效率。

AI辅助功能的优势

  • 降低学习门槛：通过自然语言交互，即使不懂代码也能理解网页逻辑
  • 提高开发效率：AI自动生成代码，节省大量编写时间
  • 智能错误修复：快速定位并修复代码错误
  • 持续优化建议：AI不断提供改进建议，帮助提升代码质量

使用技巧

  1. 明确描述需求：越详细的描述，AI生成的代码越准确
  2. 分步骤操作：复杂功能可以分解为多个小步骤，逐步实现
  3. 主动提问：遇到不理解的地方，随时向AI提问
  4. 反复尝试：如果第一次生成的效果不理想，可以调整提示词重新生成
