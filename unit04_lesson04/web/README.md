## 🚀 使用方法

### 方法1：直接打开
双击 `index.html` 文件，即可在默认浏览器中打开网页。

### 方法2：使用浏览器打开
1. 打开浏览器（推荐使用Chrome、Edge、Firefox等现代浏览器）
2. 将 `index.html` 文件拖拽到浏览器窗口中
3. 或者在浏览器中按 `Ctrl+O`，选择 `index.html` 文件打开

## 🎮 操作指南

### 导航控制
- **鼠标滚轮**：在每个章节内滚动浏览内容，到达底部/顶部后切换章节
- **键盘方向键**：↑↓ 切换章节，Home键回到首屏，End键跳到最后
- **触摸滑动**：移动设备上下滑动切换章节
- **顶部导航**：点击导航栏切换不同页面（概述/章节/作业）
- **侧边导航**：在课程章节页面，右侧导航可快速跳转到对应章节

## 📝 注意事项

1. **浏览器版本**：建议使用最新版本的现代浏览器以获得最佳体验
2. **JavaScript**：需要启用JavaScript才能正常使用交互功能
3. **图片加载**：首次打开时需要加载图片资源，请确保网络连接正常
4. **屏幕分辨率**：建议使用1920×1080或更高分辨率以获得最佳视觉效果

## 🎨 自定义修改

如需修改网页内容：
- **修改文字内容**：编辑 `index.html` 文件
- **调整样式颜色**：编辑 `style.css` 文件中的 `:root` 变量
- **修改交互行为**：编辑 `script.js` 文件
- **更换图片**：替换 `images/` 文件夹中的对应图片文件
