# 板块二：课程章节

## 一、提示词的作用

在AI视频创作中，“提示词”（Prompt）就是告诉AI你想拍什么的“语言”。如果说AI是导演，那提示词就是导演的“口令”。写得清楚，AI就能理解你想要什么；写得模糊，生成的视频就可能偏题。

一个好的提示词能让画面更有故事感、情绪感和视觉冲击力。它不仅告诉AI画面中有什么，还要让AI知道：是谁、在干什么、在哪里、气氛是什么样。

## 二、AI生成视频提示词的基本结构

写提示词其实有一个简单公式：

> 主体 + 主体动作 + 场景 + 镜头语言 + 光影 + 氛围
> 

理解这个结构后，哪怕是第一次用AI视频工具，也能轻松生成画面感强的视频。

### （一）主体（谁在画面里）

主体是视频中最重要的部分，可以是人、动物或物体。写的时候要具体，让AI清楚地知道是什么。

- 不写“一个人”，而写“一个穿红裙的女孩”。
- 不写“花”，而写“在微风中摇曳的粉色樱花”。

> 示例：一位穿白衬衫、戴眼镜的年轻男子站在街头。
> 

### （二）主体动作（在做什么）

动作让画面“动”起来，是AI视频最关键的元素之一。动得好，画面就会有生命力。

常见动作类型：

- 人物类：行走、回头微笑、奔跑、跳舞；
- 动物类：奔跑、摇尾巴、飞翔、抬头；
- 物体类：旋转、漂浮、闪烁、升起。

> 示例：一只猫在阳光下轻轻伸懒腰。
> 

### （三）场景（在哪里发生）

场景是故事的背景。一个细节丰富的场景能让视频更真实。

可以是：

- 室外：森林、海边、街道、雪地；
- 室内：教室、咖啡馆、工作室；
- 特殊环境：雨天、夜晚、梦境、未来城市。

> 示例：在雨后的东京街头，霓虹灯倒映在湿润的路面上。
> 

### （四）镜头语言（从哪个角度看）

镜头语言让画面更像电影。不同角度，会改变画面的感觉。

| 名称 | 含义 | 画面效果 | 适合场景 |
| --- | --- | --- | --- |
| **远景** | 拍得比较远，能看到整个环境和人物全身。 | 让观众了解“故事发生在哪里”。 | 展示背景、环境氛围，如城市街头、山海风光。 |
| **中景** | 拍到人物从腰部以上，既能看到人，也能看到动作。 | 画面平衡，既有情节又有空间感。 | 人物对话、走路、日常活动。 |
| **特写** | 很近地拍摄，只看到人物脸或物体的局部。 | 强调情绪、细节、质感。 | 表达人物情感、突出物品特征。 |
| **仰拍** | 摄像机在下面往上拍。 | 让人物显得高大、有气势。 | 英雄登场、建筑宏伟、震撼场面。 |
| **俯拍** | 摄像机在上面往下拍。 | 让画面更有空间感，或者让人物显得渺小。 | 拍整体布局、梦境、孤独感画面。 |
| **航拍** | 用无人机等高空设备从上往下拍。 | 视野开阔、壮观大气。 | 风景、城市全景、电影开场。 |

> 示例：镜头从低角度拍摄，女孩的裙摆在风中飘扬。
> 

### （五）光影（画面的光线感觉）

光影是营造氛围的关键。不同的光线，会带来不同的情绪。

| **光线类型** | **视觉效果** | **常见用途** |
| --- | --- | --- |
| 柔和阳光 | 温暖、浪漫 | 午后画面 |
| 金色余晖 | 怀旧、温馨 | 傍晚情景 |
| 蓝色冷光 | 科幻、夜景 | 都市未来感 |
| 明暗对比 | 紧张、悬疑 | 剧情片效果 |

> 示例：夕阳从窗外照进来，金色的光洒在女孩的侧脸上。
> 

### （六）氛围（整体情绪）

氛围让视频“有情感”。

它决定了观众“看完的感觉”——是梦幻？温馨？孤独？还是紧张？

常用的氛围词：

- 温馨类：柔和、宁静、浪漫；
- 紧张类：阴暗、神秘、压抑；
- 活力类：明亮、热闹、轻快。
- 梦幻类：朦胧、光影流动、浪漫。
- 史诗感：壮丽、恢宏、强烈对比。

> 示例：阳光透过树叶洒在花海上，氛围温暖而梦幻。
> 

## 三、AI视频脚本编写

### （一）什么是AI视频脚本

AI视频脚本是**将创意变成可执行视频指令的文字结构**。在AI创作中，脚本不再只是影视分镜的文字说明，它更像是“AI理解创意的蓝图”。一个清晰的AI脚本，可以让模型理解故事结构、镜头顺序和节奏安排，从而生成逻辑完整的视频。

与传统影视脚本不同，AI脚本强调“信息结构化”，以方便AI解析。它通常包含三个部分：

| 模块 | 说明 | 示例 |
| --- | --- | --- |
| 场景描述 | 画面背景与氛围 | “夜晚的城市街头，灯光闪烁。” |
| 动作描述 | 主体在做什么 | “女孩走在街上，撑伞抬头看天。” |
| 语气与节奏 | 画面速度、音乐感 | “节奏舒缓，情绪温柔。” |

### （二）AI脚本的写作结构

AI脚本常采用“分段式”写法，每段描述一个镜头或场景，帮助AI分清叙事节奏。

格式如下：

```
【镜头1】（场景）在傍晚的海边，天空泛着橘红色。
【动作】一个男孩坐在岩石上吹海螺。
【氛围】温柔、梦幻、安静。
```

### （三）AI脚本的故事结构

1. **三段式结构**

**结构公式：** 开场引入 → 矛盾/目标推进 → 高潮与收束

| 阶段 | 说明 | 举例：AI短片《阳光下的纸飞机》 |
| --- | --- | --- |
| **1. 开场引入** | 交代人物是谁、在哪里、环境是什么。画面要清楚、节奏平稳。 | 蓝天下的操场，一个少年站在跑道上，阳光明亮，风轻轻吹。 |
| **2. 矛盾/目标推进** | 主人公开始做一件事，或者遇到一个小目标/挑战，让画面有“动”的感觉。 | 少年认真折纸飞机，抬头看风向，准备放飞。 |
| **3. 高潮与收束** | 故事达到顶点并结束，画面出现情绪或动作的完成。 | 少年放出纸飞机，看着它飞过树梢，露出微笑。 |

就像一个小故事的起承转合：先告诉观众是谁（开场），然后展示“他在干什么”（推进），最后让结果出现（收束）。

1. **五段式结构**

**结构公式：** 起 → 承 → 转 → 合 → 收束/余韵

| 阶段 | 说明 | 示例（纸飞机） |
| --- | --- | --- |
| **1. 起** | 介绍人物、时间、地点。建立故事背景。 | 少年在操场上，阳光明亮。 |
| **2. 承** | 展示人物准备或铺垫动作。 | 他低头折纸飞机，表情专注。 |
| **3. 转** | 出现变化或转折，是画面最有冲击力的瞬间。 | 少年抛出纸飞机，它被风带上天空。 |
| **4. 合** | 动作完成，画面稳定，节奏慢下来。 | 纸飞机飞远，镜头跟随。 |
| **5. 收束/余韵** | 情绪延续或主题升华，让观众留下印象。 | 少年微笑抬头看天，阳光洒在脸上，画面慢慢淡出。 |
1. **短视频实操版结构**

钩子（0–3s）→ 信息/动作推进（4–20s）→ 亮点/转折/高潮（21–40s）→ 收束/记忆点（41–60s）

| 阶段 | 说明 | 示例（纸飞机） |
| --- | --- | --- |
| **1. 钩子（0–3秒）** | 一开始就吸引注意，让人想看下去。可以用强烈画面或动词。 | 特写：纸飞机在手中轻轻展开，阳光闪烁。 |
| **2. 推进（4–20秒）** | 交代人物和情境，展示动作过程。 | 少年折好纸飞机，跑道延伸向远方。 |
| **3. 亮点/转折/高潮（21–40秒）** | 最具情绪和动感的部分，让观众印象最深。 | 少年跑起来，把纸飞机抛向蓝天。镜头跟随上升。 |
| **4. 收束/记忆点/CTA（41–60秒）** | 给出情感收尾或引发共鸣，让观众有“结束但意犹未尽”的感觉。 | 少年停下脚步，看着纸飞机飞远，嘴角微笑。字幕出现：“有些梦想，只要放飞就足够。” |

这种结构最接近传统影视分镜逻辑，适合一分钟左右的AI视频。

### （三）AI脚本编写的关键技巧

1. **一句话只表达一个画面**
    - 避免复杂叙事，一段描述一个镜头即可。
2. **时间顺序清晰**
    - 按“起→承→转→合”的顺序写，让AI更容易理解故事逻辑。
3. **用简单句表达复杂意象**
    - 少用修辞，多用具象：
        
        > “她笑得像花” ❌
        > 
        > 
        > “她微笑着，阳光照在脸上” ✅
        > 
4. **加入节奏词或音乐感**
    - 如“慢慢走”“轻轻转身”“节奏舒缓”，AI会自动匹配镜头时长与节奏。

短视频的时间短、节奏快，重点是**前3秒抓人、20秒讲清楚、最后5秒留印象**。

AI创作时可以把这四步写进脚本或提示词，让生成逻辑更顺畅。

### （四）示例：AI短片脚本（简单版）

**题目：** 春天的告白

```
【镜头1】早晨，阳光透过树叶洒在校园小路上。
【镜头2】穿白衬衫的男孩骑着单车经过，风吹动他的头发。
【镜头3】女孩站在樱花树下，轻轻抬头。
【镜头4】两人对视，镜头缓缓拉近。
【镜头5】花瓣飘落，背景音乐温暖。
```

这个脚本可以直接输入到Sora 2中，每个镜头对应一个生成片段，再拼接成完整视频。
