01

CSS基础入门

CSS是什么

CSS（层叠样式表）是控制网页外观的语言——如果HTML是骨架，CSS就是外衣，决定网页的颜色、字体、间距与布局。

基本语法

选择器 { 属性 : 值 ; }

示例代码

h1 {
  color: blue;
  font-size: 24px;
}

解释：这段代码让网页中的一级标题文字显示为蓝色，字号为24px。

外部样式表写法（推荐）

将CSS代码写在独立的 .css 文件中，然后在HTML中引入：

<link rel="stylesheet" href="style.css">

为什么推荐？一个CSS文件可以控制多个网页，修改方便，结构清晰。

02

6个核心CSS属性

1. color - 文字颜色

color 属性控制文字的颜色，让文字更醒目或柔和。

示例代码

h1 {
  color: #f43f5e;  /* 粉红色标题 */
}

p {
  color: #333;     /* 深灰色正文 */
}

效果：标题显示为粉红色，段落文字显示为深灰色。

2. background-color - 背景颜色

background-color 设置元素的背景颜色，可以突出内容区域。

示例代码

body {
  background-color: #f5f5f5;  /* 浅灰色背景 */
}

.highlight {
  background-color: yellow;   /* 黄色高亮 */
}

效果：整个页面背景为浅灰色，高亮区域为黄色。

3. font-size - 字体大小

font-size 控制文字的大小，用于建立视觉层次。

示例代码

h1 {
  font-size: 32px;   /* 大标题 */
}

p {
  font-size: 16px;   /* 正文大小 */
}

效果：标题字号为32px，正文字号为16px，层次清晰。

4. padding - 内部空间

padding 是元素内容与边框之间的空间，让内容"呼吸"，不显拥挤。

按钮演示对比

❌ 没有 padding

文字紧贴边缘，看起来拥挤

✅ 有 padding

文字周围有空间，视觉舒适

示例代码

button {
  padding: 15px 40px;  /* 上下15px，左右40px */
}

5. margin - 外部空间

margin 是元素与周围其他元素之间的距离，控制元素间的间隔。

卡片间距演示

❌ 没有 margin
卡片1
卡片2
卡片3

卡片紧贴在一起，难以区分

✅ 有 margin
卡片1
卡片2
卡片3

卡片之间有间距，层次清晰

示例代码

.card {
  margin-bottom: 20px;  /* 卡片底部间距20px */
}

6. border-radius - 圆角

border-radius 让元素的四个角变圆，使设计更柔和、现代。

圆角演示

border-radius: 0;

直角（默认）

border-radius: 8px;

小圆角

border-radius: 20px;

中圆角

border-radius: 50%;

圆形

示例代码

button {
  border-radius: 8px;   /* 柔和的圆角 */
}

.avatar {
  border-radius: 50%;   /* 圆形头像 */
}

💡 练习：看懂代码，翻译成人话

下面是一段CSS代码，请尝试用自己的话说出它的作用：

练习题

.welcome-box {
  color: white;
  background-color: #3b82f6;
  font-size: 24px;
  padding: 30px;
  margin: 20px;
  border-radius: 15px;
}

参考答案

这段代码创建了一个欢迎框样式：
• 文字是白色
• 背景是蓝色（#3b82f6）
• 字体大小是24像素
• 内容与边框之间有30像素的内边距，让内容不拥挤
• 盒子与周围有20像素的外边距，与其他元素保持距离
• 四个角有15像素的圆角，看起来柔和美观

03

Flex布局一句话

display: flex — 让元素横向排列

Flex布局是现代网页最常用的布局方式，只需要一行代码 display: flex;，就能让元素从"上下排列"变成"左右排列"。

可视化演示

❌ 默认（竖向）
项目1
项目2
项目3
✅ Flex（横向）
项目1
项目2
项目3

示例代码

nav {
  display: flex;  /* 让导航项横向排列 */
}

效果：导航栏的链接会在一行内从左到右排列，而不是从上到下。

记住：Flex是控制"布局方向"的神器，一句代码搞定横向排列。

04

AI辅助CSS设计

为什么要学会描述（AI只懂人话）

AI不能读懂你的想法，只能听懂你的语言。

当你说"让网页好看一点"，AI无法理解——什么是"好看"？颜色？字体？间距？

但如果你说"按钮用蓝色背景、白色文字、圆角8px"，AI就能准确生成。

核心原则：描述越具体，AI生成的结果越准确。学会用CSS术语表达需求，是使用AI的关键能力。

错误vs正确的描述对比

好的描述能让AI一次到位，差的描述会让你反复修改。对比以下示例：

❌ 模糊描述

"让标题好看一点"

问题：什么是"好看"？AI无法理解

✅ 清晰描述

"标题用32px，粗体，深蓝色"

结果：AI能准确生成代码

❌ 模糊描述

"按钮要有间距"

问题：内部间距还是外部间距？

✅ 清晰描述

"按钮padding上下15px、左右30px"

结果：明确具体，AI一次成功

AI辅助设计流程

AI在网页样式设计中，最核心的作用不是"帮我们写CSS"，而是帮助我们用自然语言表达设计意图，并通过多轮反馈逐步完善样式。

1

描述需求

明确对象、风格、细节

2

分析结果

检查结构、视觉、风格

3

优化迭代

调整色彩、层次、排版

4

自检验证

对齐、统一、对比度

阶段一：描述样式需求

AI能够根据自然语言生成CSS样式，但前提是描述要具体、有方向。

1

明确对象

指出要调整的部分（如导航栏、标题区、按钮）

2

明确风格

指定整体风格特征（如清新、科技、温暖、正式）

3

明确细节

包含主色调、字体大小、边距、布局方式

4

参考说明

给出参考风格或网站示例

提示词示例

"请为一个校园图书馆网页生成清新简约风格的CSS样式，主色为蓝绿色，标题加粗，导航栏置顶，整体留白充足。"

"希望风格接近苹果官网，整体更干净，字体更细。"

阶段二：分析AI生成结果

AI输出的样式只是一个初稿，设计者要像评审作品一样去"读懂它"。

结构合理性

样式是否与HTML层级对应；类名与选择器是否规范

视觉协调性

主色与背景对比是否适中；字体大小、行距是否均衡

风格一致性

各区块是否遵循同一视觉体系；按钮、图片、标题的样式是否统一

提示：尽量不要直接使用AI生成的第一个版本，应像做"设计评审"一样，指出不协调之处并准备优化。

阶段三：优化与迭代

分析完成后，可以通过自然语言进一步优化样式。

调整色彩

"请将整体配色改为暖色调，并降低饱和度。"

控制层次

"让标题与正文层次更明显，标题加粗并增加上下间距。"

改善排版

"请在各模块之间增加留白，让页面更简洁整齐。"

统一风格

"请保持所有按钮样式一致，使用圆角和主色背景。"

AI的优势在于可快速生成多个版本，设计者可以从中选择最佳方案。这一过程可培养"审美判断"与"语言表达能力"。

阶段四：AI自检与结果验证

在完成优化后，还应让AI帮助进行样式的自我检查。

各元素对齐是否整齐

字体大小与间距是否统一

色彩对比度是否足够

模块间距是否一致

自检提示示例

"请帮我检查这段CSS是否存在颜色对比度不足或层级混乱的问题。"

"请评估此页面的文字可读性和布局平衡度，并给出优化建议。"

通过不断与AI对话，不仅能提升网页的视觉表现力，也能学会更精准地表达设计意图，训练设计思维与美学判断力。

💼 真实案例演示：从需求到成品

通过一个完整案例，看看如何用AI一步步优化按钮样式。

步骤 1

初始需求

用户说："我想要一个购买按钮"

AI提示词："请帮我设计一个'立即购买'按钮的CSS样式"

AI生成代码：

.buy-button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
}

❌ 问题：太基础，不够美观

步骤 2

优化描述

用更具体的语言描述需求

优化提示词："请设计一个现代感的购买按钮，蓝色渐变背景，白色文字，padding上下15px、左右40px，添加8px圆角和轻微阴影"

AI优化代码：

.buy-button {
  background: linear-gradient(135deg, #3b82f6, #2563eb);
  color: white;
  padding: 15px 40px;
  border-radius: 8px;
  box-shadow: 0 4px 12px rgba(59, 130, 246, 0.3);
  border: none;
  font-size: 16px;
  font-weight: 600;
}

✅ 好多了！现代、美观、专业

步骤 3

添加交互

让按钮更有交互感

进一步提示："请添加鼠标悬停效果，悬停时按钮向上移动3px，阴影加深"

AI添加交互代码：

.buy-button:hover {
  transform: translateY(-3px);
  box-shadow: 0 8px 20px rgba(59, 130, 246, 0.4);
}

🎯 完美！有交互反馈，用户体验更好

案例总结

从一个简单的需求到专业的成品按钮，关键在于：

  • 第一步：基础实现 — 告诉AI做什么
  • 第二步：具体优化 — 用CSS术语描述细节（颜色、尺寸、圆角、阴影）
  • 第三步：增强体验 — 添加交互效果

💡 与AI对话就像和设计师沟通——描述越清晰，结果越精准！
